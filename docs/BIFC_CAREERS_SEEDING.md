# BIFC Careers Seeding

Seed data should use fictional records only.

Recommended seed set:

- One super administrator
- One BIFC recruiter
- One regional manager
- Three employers
- Five employer locations
- Six job advertisements
- Twenty candidate profiles
- Qualifications and visibility variations
- Matches
- Applications
- Employer interest requests
- Candidate responses
- Messages
- Interviews
- One placement

No real trainer, employer or candidate personal data should be used in development fixtures.
